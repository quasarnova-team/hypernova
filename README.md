# hypernova

**The next era of DIP** — publish/subscribe data interchange for control
systems, built on OPC UA Pub/Sub (Part 14).

A registry that doubles as a live browser, publish/subscribe by name in five
lines of Python, native C++ publishers via [supernova](https://github.com/quasarnova-team/supernova)
servers, and explicit relays for crossing network boundaries.

See [VISION.md](VISION.md) and [ARCHITECTURE.md](ARCHITECTURE.md). Full
documentation under [doc/](doc/).
